# EduShield AI

EduShield AI is an AI-powered early warning system designed to predict student dropout
and mental health risks at an early stage.

## Problem Statement
Many students drop out due to academic pressure, stress, and lack of timely support.
Institutions often identify these issues too late.

## Solution
EduShield AI analyzes academic and behavioral data such as attendance, internal marks,
engagement, and stress levels to generate a risk score and recommend early interventions.

## Features
- AI-based risk prediction
- Explainable results (why a student is at risk)
- Personalized intervention recommendations
- Interactive Streamlit dashboard

## Technology Stack
- Python
- Scikit-learn
- Pandas
- Streamlit

## How to Run
1. Install dependencies:
   pip install streamlit pandas scikit-learn
2. Run the app:
   streamlit run app.py
